# Computer Networks Protocols

This repository showcases independent implementations of fundamental
computer networking and communication algorithms, with a focus on
error detection, error correction, and channel coding techniques.

The projects emphasize clear algorithmic design, bit-level operations,
and correctness under error conditions.
